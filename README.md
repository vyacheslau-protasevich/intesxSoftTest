# Bsw test task

## Запуск по шагам:
- Создайте `.env` файл по примеру файла `.env.example`
- Запустите docker compose:
```
docker compose up --build
```
- SwaggerUI будет доступен по адресу `http://127.0.0.1:3000/docs`

## Использованные технологии:
- Python 3.11
- FastAPI
- Pydantic
- PostgreSQL
- SqlAlchhemy & Alembic
- Docker
